tensorgrad.structure

Declarative structural identity for tensorgrad expressions.

Every node class implements a single structure() method returning a Structure — the ONLY source of structural identity. Two generic folds consume it:

  • build_graph() — the networkx multigraph used for exact isomorphism verification (VF2), isomorphisms() edge mappings, symmetry-orbit discovery and drawing.
  • canon_info() — compositional Merkle-style fingerprints (an invariant coarse hash for __hash__, a sound refined fingerprint for the is_isomorphic fast paths, and per-edge colors) — see the contract documentation below.

Having both derived from one declaration removes the drift hazard of the old design, where each class hand-built its nx graph (structural_graph) while canon.py re-encoded the same facts (symmetries, constraints, weights, ...) with its own conventions. A real instance of that drift: the old canon encoder sorted raw (Tensor, Tensor) constraint pairs, so hash() crashed with a TypeError on any Variable carrying two or more value constraints, while the graph path used the sortable _constraint_keys.

The Structure schema

Endpoints (plain tuples)::

("child", i, edge)   # `edge` of children[i]
("free", name)       # a free edge of this node
("port", role)       # an anchor on the node itself (orbit, function slot)

Structure.junctions is a frozenset of junctions; a junction (frozenset of endpoints) means "these endpoints are the same wire". Inner edge names are used only to GROUP endpoints into junctions and are then discarded, so alpha-invariance holds by construction. Free names appear only inside ("free", name) endpoints, and only the match_edges variants of the folds consume them. child_roles: children with equal roles form an interchangeable multiset (Product factors, equal-weight Sum terms); children with distinct roles are slotted (Function inputs, Derivative wrt/tensor).

Fingerprint contracts (unchanged from the old canon module)

structural_hash(t) — the invariant (coarse) hash. a.is_isomorphic(b) => structural_hash(a) == structural_hash(b). Built from multiset/WL-style encodings with no tie-breaking, so it is invariant under free-edge renaming, inner-edge renaming, term/factor reordering and graph automorphisms. Backs Tensor.__hash__.

structural_fingerprint(t) — the sound (refined) fingerprint. structural_fingerprint(a) == structural_fingerprint(b) => a.is_isomorphic(b) (up to 64-bit collisions). Where WL refinement cannot canonically order symmetric parts, deterministic tie-breaks (edge names, positions) pick an order; ties can make two isomorphic tensors get DIFFERENT fingerprints (an effectiveness loss, never a soundness loss).

Every combiner also returns a color per free edge with two invariants: (I1) equal fingerprints imply there is an isomorphism that preserves edge colors, and (I2) any color-preserving permutation of one tensor's free edges is realized by an automorphism (colors only merge edges that are independently interchangeable). (I2) lets a parent encode wiring "up to color" without losing information.

Caveats:

  • Hash values are SEED-STABLE (blake2b over a canonical repr): equal expressions get equal fingerprints across processes and PYTHONHASHSEED values, so simplify() can use them as deterministic sort keys.
  • Size keys use (name, assumptions) of a Symbol (sympy interns symbols by these, so it matches object identity in-process).
  • Unknown Tensor subclasses (no structure()): the refined fingerprint falls back to per-object identity (trivially sound); the coarse hash to (type-name, order) (trivially invariant, maximally colliding).

 99@dataclass(frozen=True)
100class Structure:
101    """Complete structural identity of one node (see module docstring).
102
103    ``label`` must be a (nested) tuple of str/int/bool — all non-child
104    identity: class tag, sizes (via :func:`size_key`), signature name,
105    constraint keys, Affine rows, ...  ``transparent`` marks a node that does
106    not exist structurally (Rename): it must have exactly one child and only
107    ``{("child", 0, old), ("free", new)}`` junctions; the folds splice the
108    child through and just relocate its free edges.
109    """
110
111    label: tuple
112    children: tuple = ()
113    child_roles: tuple = ()
114    junctions: frozenset = frozenset()
115    transparent: bool = False

118def structure_of(t: Any) -> Structure:
119    """Memoized ``t.structure()`` (tensors are immutable once constructed)."""
120    # (cast: pyright resolves instance __dict__ through the metaclass as a
121    # read-only mappingproxy; instances have a plain mutable dict)
122    d = cast(dict, t.__dict__)
123    s = d.get("_structure_v1")
124    if s is None:
125        d["_structure_v1"] = s = t.structure()
126    return s

129def _H(*args: Any) -> int:
130    """Stable 64-bit combiner: independent of PYTHONHASHSEED and process.
131
132    All arguments are (nested) tuples of str/int/bool, whose ``repr`` is a
133    canonical encoding, so blake2b over it is collision-resistant."""
134    return int.from_bytes(hashlib.blake2b(repr(args).encode(), digest_size=8).digest(), "big")

137@lru_cache(maxsize=None)
138def size_key(s: Any) -> tuple:
139    """Process-stable identity of an edge size (sympy Symbol or number).
140
141    Sympy interns Symbols by (name, assumptions), so this coincides with
142    object identity in-process while staying stable across processes."""
143    if isinstance(s, Symbol):
144        return ("sym", s.name, tuple(sorted((k, str(v)) for k, v in s.assumptions0.items())))
145    return ("num", repr(s))

152# A junction, canonicalized: (sorted port-role reprs, sorted (i, edge) child
153# endpoints, sorted free names).
154_Junc = tuple[tuple[str, ...], tuple[tuple[int, str], ...], tuple[str, ...]]
155
156
157def _prepared_junctions(s: Structure) -> list[_Junc]:
158    """Canonical (deterministic, seed-independent) list of junctions."""
159    out: list[_Junc] = []
160    for j in s.junctions:
161        childs: list[tuple[int, str]] = []
162        frees: list[str] = []
163        ports: list[str] = []
164        for ep in j:
165            kind = ep[0]
166            if kind == "child":
167                childs.append((ep[1], ep[2]))
168            elif kind == "free":
169                frees.append(ep[1])
170            else:
171                ports.append(repr(ep[1]))
172        out.append((tuple(sorted(ports)), tuple(sorted(childs)), tuple(sorted(frees))))
173    out.sort()
174    return out

177def _rename_pairs(juncs: list[_Junc]) -> list[tuple[str, str]]:
178    """(old, new) pairs of a transparent node's junctions."""
179    pairs = []
180    for ports, childs, frees in juncs:
181        assert not ports and len(childs) == 1 and len(frees) == 1, "bad transparent junction"
182        pairs.append((childs[0][1], frees[0]))
183    return pairs

191def build_graph(t: Any) -> tuple[nx.MultiDiGraph, dict[str, int]]:
192    """Build the structural graph of ``t``: one mutable graph threaded
193    through the whole fold (no per-level union/relabel copies).
194
195    Returns (G, edges) where node 0 is the root and ``edges`` maps each free
196    edge name to the node it attaches to.  Shape: root(label) -> role node ->
197    child root per child; one node per junction, receiving an edge from every
198    member child edge node, attached to the root iff it carries a port; free
199    edges resolve to their junction's node.
200    """
201    G = nx.MultiDiGraph()
202    root, edges = _emit(G, t)
203    assert root == 0
204    return G, edges
205
206
207def _emit(G: nx.MultiDiGraph, t: Any) -> tuple[int, dict[str, int]]:
208    s = structure_of(t)
209    juncs = _prepared_junctions(s)
210    if s.transparent:
211        root, inner = _emit(G, s.children[0])
212        return root, {new: inner[old] for old, new in _rename_pairs(juncs)}
213    root = G.number_of_nodes()
214    G.add_node(root, name=("N", s.label), tensor=t)
215    child_edges: list[dict[str, int]] = []
216    for child, role in zip(s.children, s.child_roles):
217        rn = G.number_of_nodes()
218        G.add_node(rn, name=("Role", repr(role)))
219        G.add_edge(root, rn)
220        c_root, c_edges = _emit(G, child)
221        G.add_edge(rn, c_root)
222        child_edges.append(c_edges)
223    edges: dict[str, int] = {}
224    for ports, childs, frees in juncs:
225        jn = G.number_of_nodes()
226        G.add_node(jn, name=("J", ports))
227        if ports:
228            G.add_edge(root, jn)
229        for i, e in childs:
230            G.add_edge(child_edges[i][e], jn)
231        for name in frees:
232            edges[name] = jn
233    return root, edges

263def refined_sort_key(t: Any) -> tuple:
264    """Deterministic (seed- and process-stable) total-order key on tensors.
265
266    Name-sensitive: two tensors get the same key only if they are isomorphic
267    *with matching edge names* (fingerprint equality plus identical
268    name->color maps realizes a name-preserving isomorphism, by (I1)+(I2)).
269    Used by simplify() to order Sum terms / Product factors independently of
270    construction order and PYTHONHASHSEED."""
271    info = canon_info(t)
272    return (info.refined_fp, tuple(sorted((e, info.refined_colors[e]) for e in t.edges)))
273

275def canon_info(t: Any) -> CanonInfo:
276    """Compute (memoized, iteratively -- no recursion limit) the CanonInfo of t."""
277    info = t.__dict__.get(_ATTR)
278    if info is not None:
279        return info
280    # Iterative post-order over the expression DAG (id-sharing respected via memo).
281    stack = [t]
282    while stack:
283        node = stack[-1]
284        if _ATTR in node.__dict__:
285            stack.pop()
286            continue
287        pending = [k for k in _children(node) if _ATTR not in k.__dict__]
288        if pending:
289            stack.extend(pending)
290            continue
291        cast(dict, node.__dict__)[_ATTR] = _compute(node)
292        stack.pop()
293    return t.__dict__[_ATTR]

303def _stable_partition_rounds(n: int, sigs: list[int], step: Callable[[list[int]], list[int]]) -> list[int]:
304    """Iterate `step` until the induced partition of range(n) stabilizes.
305
306    Round count is partition-driven, hence identical on isomorphic inputs.
307    """
308
309    def partition(s: list[int]) -> list[tuple[int, ...]]:
310        groups = defaultdict(list)
311        for i, v in enumerate(s):
312            groups[v].append(i)
313        return sorted(tuple(g) for g in groups.values())
314
315    part = partition(sigs)
316    for _ in range(n + 1):
317        new = step(sigs)
318        new_part = partition(new)
319        if new_part == part:
320            return new
321        sigs, part = new, new_part
322    return sigs
323

325def _compute(t: Any) -> CanonInfo:
326    try:
327        s = structure_of(t)
328    except NotImplementedError:
329        return _canon_opaque(t)
330    juncs = _prepared_junctions(s)
331
332    if s.transparent:
333        k = canon_info(s.children[0])
334        pairs = _rename_pairs(juncs)
335        coarse = {new: k.coarse_colors[old] for old, new in pairs}
336        refined = {new: k.refined_colors[old] for old, new in pairs}
337        return CanonInfo(k.coarse_fp, k.refined_fp, coarse, refined)
338
339    kids = [canon_info(c) for c in s.children]
340    nc, nj = len(kids), len(juncs)
341    by_child: list[list[tuple[int, str]]] = [[] for _ in range(nc)]
342    for jidx, (_ports, childs, _frees) in enumerate(juncs):
343        for i, e in childs:
344            by_child[i].append((jidx, e))
345    roles = tuple(repr(r) for r in s.child_roles)
346
347    def build(coarse: bool) -> tuple[int, dict[str, int]]:
348        fps = [k.coarse_fp if coarse else k.refined_fp for k in kids]
349        cols = [k.coarse_colors if coarse else k.refined_colors for k in kids]
350
351        # Joint WL refinement over children and junctions, seeded by child
352        # fingerprints (+roles) and junction ports (+free counts).  Free and
353        # inner edge NAMES never enter any hash.
354        seeds_c = [_H("cs", roles[i], fps[i]) for i in range(nc)]
355
356        def jstep(jsig: list[int], csig: list[int]) -> list[int]:
357            return [
358                _H(
359                    "jw",
360                    jsig[j],
361                    juncs[j][0],
362                    len(juncs[j][2]),
363                    tuple(sorted((csig[i], cols[i][e]) for i, e in juncs[j][1])),
364                )
365                for j in range(nj)
366            ]
367
368        seeds_j = [_H("js", ports, len(frees)) for ports, _childs, frees in juncs]
369        if len(set(seeds_c)) == nc:
370            # Fast path: the children are already discrete (always true for
371            # slotted roles and leaves), so no WL round can change any
372            # partition — one junction refinement is maximally informative.
373            # Iso-invariant: discreteness of the child seed partition is
374            # preserved by isomorphism, so isomorphic tensors agree on the
375            # path taken.
376            csig, jsig = seeds_c, jstep(seeds_j, seeds_c)
377        else:
378
379            def step(sigs: list[int]) -> list[int]:
380                csig = sigs[:nc]
381                new_j = jstep(sigs[nc:], csig)
382                new_c = [
383                    _H("cw", csig[i], tuple(sorted((new_j[j], cols[i][e]) for j, e in by_child[i])))
384                    for i in range(nc)
385                ]
386                return new_c + new_j
387
388            sigs = _stable_partition_rounds(nc + nj, seeds_c + seeds_j, step)
389            csig, jsig = sigs[:nc], sigs[nc:]
390
391        if coarse:
392            fp = _H("node-c", s.label, tuple(sorted(csig)), tuple(sorted(jsig)))
393            colors: dict[str, int] = {}
394            for j in range(nj):
395                frees = juncs[j][2]
396                if frees:
397                    c = _H("ce", fp, jsig[j])
398                    for name in frees:
399                        colors[name] = c
400            return fp, colors
401
402        # Refined: total canonical order over children.  Tie-breaks (child
403        # edge names, then list position) only pick the ORDER; the encoding
404        # below is a COMPLETE description of the wiring, so equal encodings
405        # imply isomorphism regardless of how ties were broken.
406        order = sorted(range(nc), key=lambda i: (csig[i], tuple(sorted(s.children[i].edges)), i))
407        rank = {i: r for r, i in enumerate(order)}
408        encs: list[tuple] = []
409        color_encs: list[tuple] = []
410        for ports, childs, frees in juncs:
411            enc = (ports, len(frees), tuple(sorted((rank[i], cols[i][e]) for i, e in childs)))
412            encs.append(enc)
413            # Singleton exception: a child whose ONLY edge ends in this
414            # port-free junction can be block-swapped with any WL-identical
415            # sibling independently of everything else, so its edge color may
416            # be sig-based instead of rank-based.  This is the broadcast
417            # `x @ Ones(...)` pattern, which matters for matching Sum terms.
418            if len(childs) == 1 and not ports and s.children[childs[0][0]].order == 1:
419                i, e = childs[0]
420                color_encs.append((ports, len(frees), (("sig", csig[i], cols[i][e]),)))
421            else:
422                color_encs.append(enc)
423        fp = _H(
424            "node-r",
425            s.label,
426            tuple((roles[i], fps[i]) for i in order),
427            tuple(sorted(encs)),
428        )
429        # Colors: junctions with identical encodings are interchangeable as
430        # blocks; sharing one color across a group is (I2)-sound when each
431        # junction contributes a single free edge (a block swap is then just
432        # a transposition of edges).  Groups of multi-free junctions (equal
433        # orbits of a Constant) get one color PER junction, tie-broken by
434        # sorted free names (the rank enters the hash, names never do).
435        groups: dict[tuple, list[int]] = defaultdict(list)
436        for j in range(nj):
437            if juncs[j][2]:
438                groups[color_encs[j]].append(j)
439        colors = {}
440        for enc, js in groups.items():
441            if len(js) == 1 or all(len(juncs[j][2]) == 1 for j in js):
442                c = _H("re", fp, enc)
443                for j in js:
444                    for name in juncs[j][2]:
445                        colors[name] = c
446            else:
447                for r, j in enumerate(sorted(js, key=lambda j: juncs[j][2])):
448                    c = _H("re2", fp, enc, r)
449                    for name in juncs[j][2]:
450                        colors[name] = c
451        return fp, colors
452
453    coarse_fp, coarse_colors = build(coarse=True)
454    refined_fp, refined_colors = build(coarse=False)
455    return CanonInfo(coarse_fp, refined_fp, coarse_colors, refined_colors)

458def _canon_opaque(t: Any) -> CanonInfo:
459    # Unknown subclass (no structure()).  Refined: unique per live object
460    # (equal only to itself -> trivially sound).  Coarse: type name + order
461    # (trivially invariant under isomorphism -> safe for __hash__).
462    coarse_fp = _H("Opaque-c", type(t).__name__, t.order)
463    refined_fp = _H("Opaque-r", id(t))
464    coarse_c = _H("opaque-e-c", coarse_fp)
465    coarse_colors = {e: coarse_c for e in t.edges}
466    refined_colors = {e: _H("opaque-e-r", refined_fp, e) for e in t.edges}
467    return CanonInfo(coarse_fp, refined_fp, coarse_colors, refined_colors)
