tensorgrad.extras.book_layout

Book-grammar layout for tensor diagrams.

Turns a tensorgrad Tensor into a diagram laid out the way the Tensor Cookbook draws them by hand. This is deliberately NOT a general graph-drawing algorithm: the book's figures follow a small layout grammar, and this module implements that grammar directly.

The grammar (see also paper/animations/SKILL.md):

  1. Spine: each connected component has a horizontal backbone -- the path between the two principal free edges (row enters left, column exits right). Nodes sit ON the line.
  2. Closures: wires not on the spine become arcs ABOVE it, nested like a chord diagram (a one-page book embedding). A crossing that cannot be nested away is drawn as a crossing -- in this notation that is meaningful.
  3. Branches: off-spine pendant structure hangs BELOW its attachment node.
  4. Free edges: horizontal stubs at the spine ends; extra free edges on interior nodes leave as short labeled stubs upward.
  5. Sums: terms side by side with +/- (weights as prefixes).
  6. Scalar factors (closed components) are juxtaposed before the open component.

Orientation convention: a Variable's declared edge order is its port order -- the FIRST edge prefers to exit left, the LAST to exit right. This is what makes covariance automatic. Callers can override with left=/right=.

The output is renderer-neutral geometry (BookLayout) plus a TikZ emitter (to_book_tikz) that speaks the book's own style vocabulary (paper/chapters/tikz-styles.tex: copydot etc.), so generated code can be pasted into the book.

The whole tensorgrad language is supported:

  • Variable / Zero -- labelled nodes (Zero is a 0 node).
  • Delta (any order) -- order 2 is an identity wire, order 0 a scalar |n|, order >= 3 a copy-dot; a closed ring of identities is a circle.
  • Product -- contraction; shared edges become wires, the rest stubs.
  • Rename -- re-keys free edges (including swaps).
  • Sum -- top level AND as parenthesized product factors (drawn as a bracketed group with wires passing through).
  • Function -- application arrows: solid for consumed edges, densely dotted for elementwise; broadcast edges stay on the argument.
  • Derivative (unevaluated) -- a Penrose loop: a fit-ellipse around the differentiated subexpression with a boundary dot and labelled whiskers for the new edges (one bends left; a pair bends left/right, like the book's \dloop/\dwhiskers). Nested derivatives nest their ellipses. A derivative whose new edge is contracted onward raises NotImplementedError -- call simplify() first.
  • Expectation -- an E[.] fit-box around the inner subexpression.

A 2-port variable forced into reversed port order (a transpose, e.g. the A^T term of a gradient) is drawn rotated 180 degrees, but only when it has a free edge -- internal contraction nodes stay upright.

Entry points (all equivalent): tensorgrad.to_book_tikz(t, **opts), t.to_book_tikz(**opts), or this module's to_book_tikz. Options: left=/right= fix a free edge's side; max_width= scales a too-wide diagram down to fit; scale= sets an explicit factor.

 128def _tex_edge(name: str) -> str:
 129    """Edge names to math: i_ -> i'; i_0_0 -> i_{0,0}; plain stays plain."""
 130    if "_" not in name:
 131        return name
 132    base, *subs = name.split("_")
 133    primes = "'" * sum(1 for p in subs if p == "")
 134    subs = [p for p in subs if p]
 135    out = base + primes
 136    if subs:
 137        out += "_{" + ",".join(subs) + "}"
 138    return out

 141def _pretty_fn(name: str) -> str:
 142    """'pow(k=-1)' -> pow_{-1}; 'exp' -> exp."""
 143    if "(" in name:
 144        base, _, arg = name.partition("(")
 145        arg = arg.rstrip(")")
 146        if "=" in arg:
 147            arg = arg.split("=", 1)[1]
 148        return rf"{base}_{{{arg}}}"
 149    return name

 152class _UnionFind(dict):
 153    def find(self, x):
 154        while self.setdefault(x, x) != x:
 155            self[x] = self[self[x]]
 156            x = self[x]
 157        return x
 158
 159    def union(self, a, b):
 160        self[self.find(a)] = self.find(b)

 163def extract_graph(tensor: Tensor) -> OpenGraph:
 164    """Flatten a (product of) tensor(s) into atoms + wires + free edges."""
 165    g = OpenGraph()
 166    uf = _UnionFind()
 167    counter = itertools.count()
 168    identity_pairs: list[tuple[str, str]] = []
 169
 170    def fresh(prefix: str) -> str:
 171        return f"__{prefix}{next(counter)}"
 172
 173    def walk(t: Tensor) -> dict[str, str]:
 174        """Return a map from t's free edge names to internal wire tokens."""
 175        if isinstance(t, Rename):
 176            inner = walk(t.tensor)
 177            # mapping: old free name -> new free name.  Re-key the exposed map.
 178            return {t.mapping.get(old, old): tok for old, tok in inner.items()}
 179        if isinstance(t, Variable):
 180            aid = len(g.atoms)
 181            toks = [fresh("w") for _ in t.edges]
 182            label = t.name
 183            g.atoms.append(AtomSpec(aid, "var", label, toks))
 184            return dict(zip(list(t.edges), toks))
 185        if isinstance(t, Zero):
 186            edges = list(t.edges)
 187            aid = len(g.atoms)
 188            if not edges:
 189                g.atoms.append(AtomSpec(aid, "scalar", "0", []))
 190                return {}
 191            toks = [fresh("w") for _ in edges]
 192            g.atoms.append(AtomSpec(aid, "var", "0", toks))
 193            return dict(zip(edges, toks))
 194        if isinstance(t, Delta):
 195            edges = list(t.edges)
 196            if len(edges) == 2:
 197                # identity: the two ends are the same wire
 198                ta, tb = fresh("w"), fresh("w")
 199                identity_pairs.append((ta, tb))
 200                return {edges[0]: ta, edges[1]: tb}
 201            if len(edges) == 0:
 202                aid = len(g.atoms)
 203                g.atoms.append(AtomSpec(aid, "scalar", str(t.size), []))
 204                return {}
 205            # order 1 (sum dot) and order >= 3 (copy dot) are both drawn as
 206            # a filled dot; order is implied by the wire count.
 207            aid = len(g.atoms)
 208            toks = [fresh("w") for _ in edges]
 209            g.atoms.append(AtomSpec(aid, "copydot", "", toks))
 210            return dict(zip(edges, toks))
 211        if isinstance(t, Product):
 212            exposed: dict[str, list[str]] = {}
 213            for f in t.factors:
 214                for name, tok in walk(f).items():
 215                    exposed.setdefault(name, []).append(tok)
 216            out: dict[str, str] = {}
 217            for name, toks in exposed.items():
 218                if len(toks) == 2:  # contraction: same wire
 219                    uf.union(toks[0], toks[1])
 220                elif len(toks) == 1:
 221                    out[name] = toks[0]
 222                else:  # pragma: no cover - Product.__init__ rejects this
 223                    raise ValueError(f"edge {name} has multiplicity {len(toks)}")
 224            return out
 225        if isinstance(t, Function):
 226            aid = len(g.atoms)
 227            fn = AtomSpec(aid, "func", _pretty_fn(t.signature.name), [])
 228            g.atoms.append(fn)
 229            out: dict[str, str] = {}
 230            for inp, consumed in zip(t.inputs, t.signature.inputs):
 231                first_atom = len(g.atoms)
 232                sub = walk(inp)
 233                elementwise = not consumed
 234                for e in consumed:
 235                    # consumed edge: a real wire into the function, drawn
 236                    # with a (solid) application arrowhead
 237                    tok = sub.pop(e)
 238                    port = fresh("w")
 239                    fn.ports.append(port)
 240                    uf.union(tok, port)
 241                    g.arrows[port] = "solid"
 242                    g.arrow_heads[port] = fn.id
 243                if elementwise:
 244                    anchor = first_atom if first_atom < len(g.atoms) else None
 245                    if anchor is None and sub:
 246                        # the argument produced no atom (a bare identity /
 247                        # pass-through wire, e.g. exp(I)): materialize a copy-dot
 248                        # anchor carrying its edges, so the function isn't left
 249                        # orphaned and disconnected from its data
 250                        anchor = len(g.atoms)
 251                        anchor_toks = []
 252                        for e in list(sub):
 253                            atok = fresh("w")
 254                            uf.union(sub[e], atok)
 255                            anchor_toks.append(atok)
 256                        g.atoms.append(AtomSpec(anchor, "copydot", "", anchor_toks))
 257                    if anchor is not None:
 258                        # synthesize the dotted application arrow from the
 259                        # input's principal atom into the function
 260                        ta, tb = fresh("w"), fresh("w")
 261                        fn.ports.append(ta)
 262                        g.atoms[anchor].ports.append(tb)
 263                        uf.union(ta, tb)
 264                        g.arrows[ta] = "dotted"
 265                        g.arrow_heads[ta] = fn.id
 266                out.update(sub)  # broadcast edges pass through by name
 267            for e in t.shape_out:
 268                port = fresh("w")
 269                fn.ports.append(port)
 270                out[e] = port
 271            return out
 272        if isinstance(t, Derivative):
 273            first_atom = len(g.atoms)
 274            sub = walk(t.tensor)
 275            enclosed = list(range(first_atom, len(g.atoms)))
 276            new_edges = [e for e in t.edges if e not in t.tensor.edges]
 277            out = dict(sub)
 278            names = []
 279            for e in new_edges:
 280                tok = fresh("w")
 281                g.deriv_tokens.add(tok)
 282                out[e] = tok
 283                names.append(e)
 284            g.derivs.append((enclosed, names))
 285            return out
 286        if Expectation and isinstance(t, Expectation):
 287            first_atom = len(g.atoms)
 288            sub = walk(t.tensor)
 289            enclosed = list(range(first_atom, len(g.atoms)))
 290            g.boxes.append((enclosed, "E"))
 291            return sub
 292        if isinstance(t, Sum):
 293            # a Sum factor becomes a parenthesized group atom; its inner
 294            # terms are laid out recursively at layout time
 295            aid = len(g.atoms)
 296            toks = [fresh("w") for _ in t.edges]
 297            atom = AtomSpec(aid, "group", "", toks, sub=t)
 298            atom.port_names = dict(zip(toks, list(t.edges)))
 299            g.atoms.append(atom)
 300            return dict(zip(list(t.edges), toks))
 301        raise NotImplementedError(f"book_layout does not support {type(t).__name__} yet")
 302
 303    exposed = walk(tensor)
 304    for ta, tb in identity_pairs:
 305        uf.union(ta, tb)
 306    g.arrows = {uf.find(k): v for k, v in g.arrows.items()}
 307    g.arrow_heads = {uf.find(k): v for k, v in g.arrow_heads.items()}
 308    for atom in g.atoms:
 309        atom.port_names = {uf.find(k): v for k, v in atom.port_names.items()}
 310
 311    # Build wires from union-find classes.
 312    endpoints: dict[str, list[int]] = {}
 313    for atom in g.atoms:
 314        atom.ports = [uf.find(p) for p in atom.ports]
 315        for p in atom.ports:
 316            endpoints.setdefault(p, []).append(atom.id)
 317    free_names: dict[str, str] = {name: uf.find(tok) for name, tok in exposed.items()}
 318
 319    for wid, ends in endpoints.items():
 320        if len(ends) > 2:
 321            raise ValueError("wire with more than two endpoints (use Delta)")
 322        g.wires[wid] = ends
 323    for name, wid in free_names.items():
 324        if wid in endpoints:
 325            g.free_wires.setdefault(wid, []).append(name)
 326        # a wire with no atoms at all: bare identity between two free ends
 327    # bare identity wires: group free names by wire id with no endpoints
 328    deriv_reps = {uf.find(tok) for tok in g.deriv_tokens}
 329    bare: dict[str, list[str]] = {}
 330    for name, wid in free_names.items():
 331        if wid not in endpoints and wid not in deriv_reps:
 332            bare.setdefault(wid, []).append(name)
 333    for wid, names in bare.items():
 334        if len(names) == 2:
 335            g.bare_wires.append((names[0], names[1]))
 336        else:  # a dangling free edge with no atom: nothing to draw it from
 337            raise ValueError(f"free edge {names} attached to nothing")
 338    for tok in g.deriv_tokens:
 339        if uf.find(tok) in endpoints:
 340            raise NotImplementedError(
 341                "a derivative's new edge is contracted onward; simplify() "
 342                "first (drawing wires from the loop boundary is not "
 343                "supported yet)"
 344            )
 345    # closed rings of pure order-2 identities: no atoms, no free names --
 346    # invisible to the loops above, but each class is a scalar circle
 347    ring_classes = set()
 348    for ta, tb in identity_pairs:
 349        rep = uf.find(ta)
 350        if rep not in endpoints and rep not in set(free_names.values()):
 351            ring_classes.add(rep)
 352    g.rings = len(ring_classes)
 353    return g

 399def _label_halfwidth(label: str) -> float:
 400    # A label may carry a TeX subscript (pow_{-1}): the base word sets most
 401    # of the width, and the subscript adds a bit more (rendered smaller).
 402    # Dropping the subscript entirely made pow_{-k} nodes far too narrow, so
 403    # their application arrows had no room and jammed against the glyph.
 404    base, sep, sub = label.partition("_{")
 405    sub = sub.rstrip("}") if sep else ""
 406    width = CHAR_W * max(0, len(base) - 1)
 407    width += 0.6 * CHAR_W * len(sub.replace("{", "").replace("}", ""))
 408    return 0.12 + width / 2

 411def _adjacency(g: OpenGraph) -> dict[int, dict[int, list[str]]]:
 412    adj: dict[int, dict[int, list[str]]] = {a.id: {} for a in g.atoms}
 413    for wid, ends in g.wires.items():
 414        if len(ends) == 2 and ends[0] != ends[1]:
 415            a, b = ends
 416            adj[a].setdefault(b, []).append(wid)
 417            adj[b].setdefault(a, []).append(wid)
 418    return adj

 421def _components(g: OpenGraph) -> list[list[int]]:
 422    adj = _adjacency(g)
 423    seen: set[int] = set()
 424    comps = []
 425    for a in g.atoms:
 426        if a.id in seen:
 427            continue
 428        stack, comp = [a.id], []
 429        seen.add(a.id)
 430        while stack:
 431            v = stack.pop()
 432            comp.append(v)
 433            for u in adj[v]:
 434                if u not in seen:
 435                    seen.add(u)
 436                    stack.append(u)
 437        comps.append(comp)
 438    return comps

 444def _simple_paths(adj: dict[int, dict[int, list[str]]], comp: list[int]) -> list[list[int]]:
 445    """Simple vertex paths within a component, deduped by orientation and
 446    capped (dense components would otherwise blow up exponentially)."""
 447    paths: list[list[int]] = []
 448
 449    def dfs(path: list[int], seen: set[int]) -> None:
 450        if len(paths) >= _PATH_CAP:
 451            return
 452        if path[0] <= path[-1]:  # each path once; scoring tries both directions
 453            paths.append(list(path))
 454        for u in adj[path[-1]]:
 455            if u in seen or u not in compset:
 456                continue
 457            seen.add(u)
 458            path.append(u)
 459            dfs(path, seen)
 460            path.pop()
 461            seen.remove(u)
 462
 463    compset = set(comp)
 464    for v in comp:
 465        dfs([v], {v})
 466    return paths

 475            for name in g.free_wires[wid]:

 480def _crossings(chords: list[tuple[int, int]]) -> int:
 481    n = 0
 482    for (a1, b1), (a2, b2) in itertools.combinations(chords, 2):
 483        if a1 < a2 < b1 < b2 or a2 < a1 < b2 < b1:
 484            n += 1
 485    return n

 488def _spine_chords(
 489    g: OpenGraph, path: list[int], adj: dict[int, dict[int, list[str]]]
 490) -> list[tuple[int, int, str]]:
 491    """Wires drawn as arcs, as (i, j, wire_id) with spine indices i <= j."""
 492    pos = {v: i for i, v in enumerate(path)}
 493    used_link: set[str] = set()
 494    # mark one connecting wire per adjacent pair as the spine segment
 495    for i in range(len(path) - 1):
 496        wids = adj[path[i]].get(path[i + 1], [])
 497        if wids:
 498            used_link.add(wids[0])
 499    chords = []
 500    for wid, ends in g.wires.items():
 501        if wid in used_link or len(ends) != 2:
 502            continue
 503        a, b = ends
 504        if a in pos and b in pos:
 505            i, j = sorted((pos[a], pos[b]))
 506            chords.append((i, j, wid))
 507    return chords

 510def _choose_spine(
 511    g: OpenGraph,
 512    comp: list[int],
 513    left: Optional[str],
 514    right: Optional[str],
 515) -> list[int]:
 516    adj = _adjacency(g)
 517    best, best_score = None, None
 518    for path in _simple_paths(adj, comp):
 519        for cand in (path, path[::-1]):
 520            score = 0.0
 521            head, tail = cand[0], cand[-1]
 522            head_frees = _atom_free_names(g, head)
 523            tail_frees = _atom_free_names(g, tail)
 524            if left is not None and any(n == left for _, n in head_frees):
 525                score += 1000
 526            if right is not None and any(n == right for _, n in tail_frees):
 527                score += 1000
 528            if head_frees:
 529                score += 100
 530                # the head's free edge prefers to be the FIRST declared port
 531                if any(idx == 0 for idx, _ in head_frees):
 532                    score += 5
 533            if tail_frees and tail != head:
 534                score += 100
 535                nports = len(g.atoms[tail].ports)
 536                if any(idx == nports - 1 for idx, _ in tail_frees):
 537                    score += 5
 538            elif tail == head and len(head_frees) >= 2:
 539                score += 100  # one node, two horizontal frees: both ends served
 540            score += 10 * len(cand)
 541            chords3 = _spine_chords(g, cand, adj)
 542            chords = [(i, j) for i, j, _ in chords3]
 543            score -= 3 * _crossings(chords)
 544            # prefer arcs that span far (trace closing over everything)
 545            score += 0.1 * sum(j - i for i, j in chords)
 546            # a closed component's spine should be part of its cycle: bonus
 547            # when a leftover wire joins the two spine endpoints (trace dome)
 548            if any(i == 0 and j == len(cand) - 1 for i, j in chords):
 549                score += 50
 550            # read in expression order (A B C D, not A D C B)
 551            score += 0.5 * sum(
 552                1 for u, w in zip(cand, cand[1:]) if u < w
 553            )
 554            # a function sits LEFT of its argument (its application arrow
 555            # then points leftward into it, as the book draws it)
 556            for u, w in zip(cand, cand[1:]):
 557                for wid in adj[u].get(w, []):
 558                    if g.arrows.get(wid):
 559                        head_atom = g.arrow_heads.get(wid)
 560                        score += 6 if head_atom == u else -6
 561            score -= 0.01 * cand[0]
 562            if best_score is None or score > best_score:
 563                best, best_score = list(cand), score
 564    assert best is not None
 565    return best

 568def _assign_lanes(chords: list[tuple[int, int, str]]) -> dict[str, int]:
 569    """Nest arcs: smaller intervals get lower lanes; conflicts stack up."""
 570    lanes: dict[str, int] = {}
 571    placed: list[tuple[int, int, int]] = []  # (i, j, lane)
 572    for i, j, wid in sorted(chords, key=lambda c: (c[1] - c[0], c[0])):
 573        lane = 1
 574        while True:
 575            ok = True
 576            for pi, pj, pl in placed:
 577                if pl != lane:
 578                    continue
 579                # same-lane conflict unless strictly disjoint; identical
 580                # intervals (double traces) always conflict
 581                if (i, j) == (pi, pj) or not (j <= pi or pj <= i):
 582                    ok = False
 583                    break
 584            # also: an arc strictly containing a placed arc must be above it
 585            for pi, pj, pl in placed:
 586                if i <= pi and pj <= j and lane <= pl and (i, j) != (pi, pj):
 587                    ok = False
 588                    break
 589            if ok:
 590                break
 591            lane += 1
 592        lanes[wid] = lane
 593        placed.append((i, j, lane))
 594    return lanes

 597def _layout_component(
 598    g: OpenGraph,
 599    comp: list[int],
 600    layout: BookLayout,
 601    x0: float,
 602    left: Optional[str],
 603    right: Optional[str],
 604    solo_side: str = "left",
 605) -> float:
 606    """Lay out one component starting at x0; return its rightmost extent."""
 607    spine = _choose_spine(g, comp, left, right)
 608    pos_index = {v: i for i, v in enumerate(spine)}
 609    adj = _adjacency(g)
 610
 611    # -- group atoms: lay out their inner Sum first (width matters) --
 612    subs: dict[int, "BookLayout"] = {}
 613    for k, v in enumerate(spine):
 614        atom = g.atoms[v]
 615        if atom.kind != "group":
 616            continue
 617        le = re = None
 618        if k > 0:
 619            wids = adj[spine[k - 1]].get(v, [])
 620            if wids:
 621                le = atom.port_names.get(wids[0])
 622        if k < len(spine) - 1:
 623            wids = adj[v].get(spine[k + 1], [])
 624            if wids:
 625                re = atom.port_names.get(wids[0])
 626        subs[v] = layout_any(atom.sub, left=le, right=re)
 627    # group atoms that land OFF the spine (as pendants) still need their
 628    # inner layout computed, or emission crashes on `assert n.sub is not None`
 629    for v in comp:
 630        if g.atoms[v].kind == "group" and v not in subs:
 631            subs[v] = layout_any(g.atoms[v].sub)
 632
 633    def halfwidth(v: int) -> float:
 634        atom = g.atoms[v]
 635        if atom.kind == "group":
 636            return subs[v].xmax / 2 + 0.36  # parens + symmetric padding
 637        if atom.kind == "copydot":
 638            return 0.06
 639        return _label_halfwidth(atom.label)
 640
 641    # -- build the below-spine pendant forest FIRST, so its subtree widths
 642    # can widen the spine: two adjacent spine nodes that each carry pendant
 643    # structure must be far enough apart that their forests don't overlap --
 644    PEND_GAP = 0.42
 645    placed = set(spine)
 646    children: dict[int, list[int]] = {}
 647    parent_of: dict[int, int] = {}
 648    parent_wire: dict[int, str] = {}
 649    _frontier = list(spine)
 650    while _frontier:
 651        u = _frontier.pop(0)
 652        for v in sorted(adj[u]):
 653            if v in placed:
 654                continue
 655            placed.add(v)
 656            children.setdefault(u, []).append(v)
 657            parent_of[v] = u
 658            parent_wire[v] = adj[u][v][0]
 659            _frontier.append(v)
 660
 661    def _pw(v: int) -> float:
 662        atom = g.atoms[v]
 663        if atom.kind == "group":
 664            return 2 * halfwidth(v)
 665        return 0.12 if atom.kind == "copydot" else 2 * _label_halfwidth(atom.label)
 666
 667    subtree_w: dict[int, float] = {}
 668
 669    def subtree_width(v: int) -> float:
 670        if v in subtree_w:
 671            return subtree_w[v]
 672        kids = children.get(v, [])
 673        w = _pw(v)
 674        if kids:
 675            kids_w = sum(subtree_width(k) for k in kids) + PEND_GAP * (len(kids) - 1)
 676            w = max(w, kids_w)
 677        subtree_w[v] = w
 678        return w
 679
 680    def forest_half(u: int) -> float:
 681        kids = children.get(u, [])
 682        if not kids:
 683            return 0.0
 684        total = sum(subtree_width(k) for k in kids) + PEND_GAP * (len(kids) - 1)
 685        return total / 2
 686
 687    # A single-node component with ONE free edge points it to solo_side only,
 688    # so it reserves stub space on just that side (an outer product's two
 689    # vectors were each reserving both sides -> a big empty gap between them).
 690    single = len(spine) == 1
 691    n_head_frees = len(_atom_free_names(g, spine[0]))
 692    reserve_left = bool(n_head_frees) and (
 693        not single or n_head_frees > 1 or solo_side == "left"
 694    )
 695    reserve_right = bool(_atom_free_names(g, spine[-1])) and (
 696        not single or n_head_frees > 1 or solo_side == "right"
 697    )
 698
 699    # -- x positions along the spine (label-width aware, pendant-aware) --
 700    xs: list[float] = []
 701    x = x0
 702    if reserve_left:
 703        x += STUB + 0.05
 704    x += max(0.0, halfwidth(spine[0]) - 0.12)  # wide heads (groups) start inside
 705    prev_half = 0.0
 706    for k, v in enumerate(spine):
 707        half = halfwidth(v)
 708        if k > 0:
 709            # an application arrow between two glyphs needs a wider clear
 710            # zone than a plain wire: room for the arrowhead AND its dots
 711            wids = adj[spine[k - 1]].get(v, [])
 712            clearance = 0.58 if any(g.arrows.get(w) for w in wids) else 0.38
 713            gap = max(PITCH, prev_half + half + clearance)
 714            # widen so neighbouring pendant forests don't interleave
 715            gap = max(gap, forest_half(spine[k - 1]) + forest_half(v) + 0.3)
 716            x += gap
 717        xs.append(x)
 718        prev_half = half
 719    for v, xv in zip(spine, xs):
 720        atom = g.atoms[v]
 721        layout.nodes.append(
 722            LNode(v, atom.kind, atom.label, xv, 0.0,
 723                  sub=subs.get(v), width=2 * halfwidth(v))
 724        )
 725
 726    drawn: list[str] = []  # every wid rendered; audited for conservation below
 727
 728    # -- spine segments (application arrows point into their function) --
 729    for k in range(len(spine) - 1):
 730        wids = adj[spine[k]].get(spine[k + 1])
 731        if wids:
 732            wid = wids[0]
 733            drawn.append(wid)
 734            arrow = g.arrows.get(wid, "")
 735            a, b = spine[k], spine[k + 1]
 736            if arrow and g.arrow_heads.get(wid) == a:
 737                a, b = b, a
 738            layout.wires.append(LWire("segment", a, b, arrow=arrow))
 739
 740    # -- arcs (keep application-arrow decorations; tip at the function) --
 741    chords = _spine_chords(g, spine, adj)
 742    lanes = _assign_lanes(chords)
 743    drawn.extend(lanes)
 744    for i, j, wid in chords:
 745        arrow = g.arrows.get(wid, "")
 746        tip = ""
 747        if arrow:
 748            tip = "->" if g.arrow_heads.get(wid) == spine[j] else "<-"
 749        if i == j:
 750            layout.wires.append(LWire("loop", spine[i], spine[i], lane=lanes[wid]))
 751        else:
 752            layout.wires.append(
 753                LWire("arc", spine[i], spine[j], direction=tip, arrow=arrow,
 754                      span=j - i, lane=lanes[wid])
 755            )
 756
 757    # -- pendants (off-spine structure hangs below its attachment) --
 758    # The forest (children/parent_of/subtree_width) was built above so the
 759    # spine could reserve room for it; now lay each subtree out recursively:
 760    # a subtree reserves its full width so siblings never overlap, and every
 761    # parent is centered over its children.
 762    node_x = {n.id: n.x for n in layout.nodes}
 763
 764    def place_subtree(v: int, cx: float, y: float) -> None:
 765        atom = g.atoms[v]
 766        layout.nodes.append(
 767            LNode(v, atom.kind, atom.label, cx, y,
 768                  sub=subs.get(v),
 769                  width=2 * halfwidth(v) if atom.kind == "group" else 0.0)
 770        )
 771        wid = parent_wire[v]
 772        drawn.append(wid)
 773        arrow = g.arrows.get(wid, "")
 774        pa, pb = parent_of[v], v
 775        if arrow and g.arrow_heads.get(wid) == pa:
 776            pa, pb = pb, pa
 777        layout.wires.append(LWire("pendant", pa, pb, arrow=arrow))
 778        node_x[v] = cx
 779        kids = children.get(v, [])
 780        if not kids:
 781            return
 782        total = sum(subtree_width(k) for k in kids) + PEND_GAP * (len(kids) - 1)
 783        x = cx - total / 2
 784        for k in kids:
 785            w = subtree_width(k)
 786            place_subtree(k, x + w / 2, y - PENDANT_DY)
 787            x += w + PEND_GAP
 788
 789    for u in spine:
 790        kids = children.get(u, [])
 791        if not kids:
 792            continue
 793        ux = node_x[u]
 794        total = sum(subtree_width(k) for k in kids) + PEND_GAP * (len(kids) - 1)
 795        x = ux - total / 2
 796        for k in kids:
 797            w = subtree_width(k)
 798            place_subtree(k, x + w / 2, -PENDANT_DY)
 799            x += w + PEND_GAP
 800
 801    # -- wire conservation: anything not yet drawn becomes an extra
 802    # connector (below the spine) or a loop -- NEVER silently dropped --
 803    from collections import Counter
 804
 805    want = Counter(
 806        wid for wid, ends in g.wires.items()
 807        if len(ends) == 2 and ends[0] in placed and ends[1] in placed
 808        for _ in range(g.atoms[ends[0]].ports.count(wid) // 2
 809                       if ends[0] == ends[1] else 1)
 810    )
 811    have = Counter(drawn)
 812    for wid, cnt in want.items():
 813        missing = cnt - have.get(wid, 0)
 814        for extra_i in range(missing):
 815            a, b = g.wires[wid]
 816            arrow = g.arrows.get(wid, "")
 817            tip = ""
 818            if arrow:
 819                tip = "->" if g.arrow_heads.get(wid) == b else "<-"
 820            if a == b:
 821                layout.wires.append(LWire("loop", a, a, lane=2 + extra_i))
 822            else:
 823                layout.wires.append(
 824                    LWire("extra", a, b, direction=tip, arrow=arrow)
 825                )
 826
 827    # -- free-edge stubs --
 828    stub_ports: dict[int, dict[str, str]] = {}
 829    for k, v in enumerate(spine):
 830        frees = _atom_free_names(g, v)
 831        if not frees:
 832            continue
 833        frees.sort()  # port order
 834        assigned: list[tuple[str, str]] = []  # (direction, name)
 835        if k == 0 and k == len(spine) - 1:  # single-node spine
 836            if len(frees) == 1:
 837                assigned.append((solo_side, frees[0][1]))
 838            else:
 839                names = [nm for _, nm in frees]
 840                lpick = names.index(left) if left in names else 0
 841                rest = [m for m in range(len(names)) if m != lpick]
 842                rpick = (names.index(right)
 843                         if right in names and names.index(right) != lpick
 844                         else rest[-1])
 845                assigned.append(("left", names[lpick]))
 846                assigned.append(("right", names[rpick]))
 847                for m, nm in enumerate(names):
 848                    if m not in (lpick, rpick):
 849                        assigned.append(("up", nm))
 850        elif k == 0:
 851            pick = 0
 852            if left is not None:
 853                for m, (_, nm) in enumerate(frees):
 854                    if nm == left:
 855                        pick = m
 856            assigned.append(("left", frees[pick][1]))
 857            for m, (_, name) in enumerate(frees):
 858                if m != pick:
 859                    assigned.append(("up", name))
 860        elif k == len(spine) - 1:
 861            pick = len(frees) - 1
 862            if right is not None:
 863                for m, (_, nm) in enumerate(frees):
 864                    if nm == right:
 865                        pick = m
 866            assigned.append(("right", frees[pick][1]))
 867            for m, (_, name) in enumerate(frees):
 868                if m != pick:
 869                    assigned.append(("up", name))
 870        else:
 871            for _, name in frees:
 872                assigned.append(("up", name))
 873        ups = [nm for d, nm in assigned if d == "up"]
 874        ui = 0
 875        for direction, name in assigned:
 876            lab = name if direction in ("up", "down") else ""
 877            off = 0.0
 878            if direction == "up" and len(ups) > 1:
 879                off = (ui - (len(ups) - 1) / 2) * 0.26
 880                ui += 1
 881            layout.wires.append(
 882                LWire("stub", v, direction=direction, label=lab, x=off))
 883            if direction == "left" and layout.left_edge is None:
 884                layout.left_edge = name
 885            if direction == "right":
 886                layout.right_edge = name
 887            stub_ports.setdefault(v, {})[direction] = name
 888
 889    # -- transpose detection: a 2-port var whose left-facing port comes
 890    # AFTER its right-facing port in declared order is drawn rotated 180
 891    # degrees (the book's transpose convention) --
 892    name_to_wid = {}
 893    for v in comp:
 894        for wid in g.atoms[v].ports:
 895            for nm in g.free_wires.get(wid, []):
 896                name_to_wid[nm] = wid
 897    for k, v in enumerate(spine):
 898        atom = g.atoms[v]
 899        real_ports = [p for p in atom.ports if p not in g.arrows]
 900        if atom.kind != "var" or len(real_ports) != 2:
 901            continue
 902        left_wid = right_wid = None
 903        if k > 0:
 904            wids = adj[spine[k - 1]].get(v, [])
 905            left_wid = wids[0] if wids else None
 906        elif stub_ports.get(v, {}).get("left") is not None:
 907            left_wid = name_to_wid.get(stub_ports[v]["left"])
 908        if k < len(spine) - 1:
 909            wids = adj[v].get(spine[k + 1], [])
 910            right_wid = wids[0] if wids else None
 911        elif stub_ports.get(v, {}).get("right") is not None:
 912            right_wid = name_to_wid.get(stub_ports[v]["right"])
 913        if left_wid is None or right_wid is None or left_wid == right_wid:
 914            continue
 915        try:
 916            li = real_ports.index(left_wid)
 917            ri = real_ports.index(right_wid)
 918        except ValueError:
 919            continue
 920        # Rotation marks a transpose, which is a boundary/covariance concept:
 921        # only meaningful when at least one port is a FREE edge whose side we
 922        # are tracking (e.g. A^T in a gradient term). A matrix contracted on
 923        # both sides has an arbitrary internal orientation -- rotating it just
 924        # produces a confusing upside-down glyph (a chain's internal K read as
 925        # a backwards R), so leave those upright.
 926        has_free = any(
 927            len(g.wires.get(p, [])) == 1 for p in real_ports
 928        )
 929        if li > ri and has_free:
 930            node = next(n for n in layout.nodes if n.id == v)
 931            node.rotated = True
 932
 933    # -- double-edge transpose: two adjacent 2-port matrices that share BOTH
 934    # their edges are either Tr(AB) (ports pair CROSSED: A's inner index meets
 935    # B's inner index) or Tr(AB^T) / ||A||_F (ports pair PARALLEL: same index
 936    # on both). A parallel pairing must rotate the right matrix, else the two
 937    # genuinely different tensors would draw byte-identically (a segment + arc
 938    # with no crossing) -- the semantic-fidelity bug. --
 939    rotated_ids = {n.id for n in layout.nodes if n.rotated}
 940    for k in range(len(spine) - 1):
 941        u, v = spine[k], spine[k + 1]
 942        shared = adj[u].get(v, [])
 943        au, av = g.atoms[u], g.atoms[v]
 944        if (len(shared) != 2 or au.kind != "var" or av.kind != "var"
 945                or len(au.ports) != 2 or len(av.ports) != 2):
 946            continue
 947        try:
 948            parallel = all(au.ports.index(w) == av.ports.index(w) for w in shared)
 949        except ValueError:
 950            continue
 951        if parallel and v not in rotated_ids:
 952            node = next(n for n in layout.nodes if n.id == v)
 953            node.rotated = True
 954            rotated_ids.add(v)
 955
 956    # free edges on pendants leave downward
 957    for v in comp:
 958        if v in pos_index:
 959            continue
 960        for idx, name in _atom_free_names(g, v):
 961            layout.wires.append(LWire("stub", v, direction="down", label=name))
 962
 963    xmax = xs[-1] + halfwidth(spine[-1])
 964    if reserve_right:
 965        xmax += STUB + 0.05
 966    return xmax

 969def layout_tensor(
 970    tensor: Tensor, left: Optional[str] = None, right: Optional[str] = None
 971) -> BookLayout:
 972    """Renderer-neutral book-grammar layout of a single (non-Sum) tensor."""
 973    g = extract_graph(tensor)
 974    layout = BookLayout()
 975    x = 0.0
 976    # bare identity wires and identity rings first
 977    for _ in g.bare_wires:
 978        layout.wires.append(LWire("bare", x=x, y=0.0))
 979        x += 1.0 + COMPONENT_GAP
 980    for _ in range(g.rings):
 981        layout.wires.append(LWire("ring", x=x + 0.28, y=0.0))
 982        x += 0.56 + COMPONENT_GAP
 983    comps = _components(g)
 984
 985    def has_frees(comp: list[int]) -> bool:
 986        return any(_atom_free_names(g, v) for v in comp)
 987
 988    def comp_order(comp: list[int]) -> tuple:
 989        if not has_frees(comp):
 990            return (0, 0)  # closed scalar factors first
 991        names = {nm for v in comp for _, nm in _atom_free_names(g, v)}
 992        if left is not None and left in names:
 993            return (1, 0)
 994        if right is not None and right in names:
 995            return (1, 2)
 996        return (1, 1)
 997
 998    comps.sort(key=comp_order)
 999    open_comps = [c for c in comps if has_frees(c)]
1000    for comp in comps:
1001        is_open = has_frees(comp)
1002        l = left if is_open else None
1003        r = right if is_open else None
1004        # first open component points its lone stub left, the last right
1005        solo = "left"
1006        if is_open and open_comps and comp is open_comps[-1] and len(open_comps) > 1:
1007            solo = "right"
1008        x_end = _layout_component(g, comp, layout, x, l, r, solo_side=solo)
1009        x = x_end + COMPONENT_GAP
1010    layout.xmax = x - COMPONENT_GAP
1011    layout.derivs = list(g.derivs)
1012    layout.boxes = list(g.boxes)
1013    return layout

1016def layout_any(
1017    tensor: Tensor, left: Optional[str] = None, right: Optional[str] = None
1018) -> BookLayout:
1019    """Layout for any supported tensor, including a top-level Sum."""
1020    if not isinstance(tensor, Sum):
1021        return layout_tensor(tensor, left, right)
1022    out = BookLayout()
1023    x = 0.0
1024    sign_id = -1  # negative ids never collide with term/atom ids
1025    for idx, (term, weight) in enumerate(zip(tensor.terms, tensor.weights)):
1026        sign, coeff = _fmt_weight(weight)
1027        show_sign = idx > 0 or sign == "-" or coeff
1028        if show_sign:

1092def _emit_layout(layout: BookLayout, lines: list[str], prefix: str, dx: float) -> None:
1093    name = {n.id: f"{prefix}n{n.id}" for n in layout.nodes}
1094    nodes = {n.id: n for n in layout.nodes}
1095    group_side: dict[int, tuple[str, str]] = {}
1096    for n in layout.nodes:
1097        x = n.x + dx
1098        if n.kind == "copydot":
1099            lines.append(rf"\node[copydot] ({name[n.id]}) at ({x:.2f},{n.y:.2f}) {{}};")
1100        elif n.kind == "group":
1101            # parenthesized inner sum: two paren nodes + recursive emission
1102            hw = n.width / 2
1103            pl, pr = f"{name[n.id]}L", f"{name[n.id]}R"
1104            lines.append(
1105                rf"\node[scale=1.45, inner sep=0.5pt] ({pl}) at ({x - hw + 0.12:.2f},{n.y:.2f}) {{$($}};"
1106            )
1107            lines.append(
1108                rf"\node[scale=1.45, inner sep=0.5pt] ({pr}) at ({x + hw - 0.12:.2f},{n.y:.2f}) {{$)$}};"
1109            )
1110            group_side[n.id] = (pl, pr)
1111            assert n.sub is not None
1112            # extra breathing room after the '(' before the first term's stub
1113            _emit_layout(n.sub, lines, prefix=f"{name[n.id]}i", dx=x - hw + 0.34)
1114        elif n.kind == "sign":
1115            lines.append(
1116                rf"\node[{_AXIS}] ({name[n.id]}) at ({x:.2f},{n.y:.2f}) {{${n.label}$}};"
1117            )
1118        else:
1119            rot = "rotate=180, " if n.rotated else ""
1120            # _AXIS pins a uniform text height/depth so a subscripted label
1121            # (pow_{-1}) centers on the same axis as exp/x -- its subscript no
1122            # longer drags the node's centre down out of line with the wire.
1123            lines.append(
1124                rf"\node[{rot}{_AXIS}] ({name[n.id]}) at ({x:.2f},{n.y:.2f})"
1125                rf" {{{_tex_label(n.label)}}};"
1126            )
1127
1128    def endpoint(nid: int, other: Optional[int]) -> str:
1129        # segments touching a group attach to the nearer paren
1130        if nid in group_side and other is not None:
1131            pl, pr = group_side[nid]
1132            return pl if nodes[other].x < nodes[nid].x else pr
1133        return name[nid]
1134
1135    # application arrows keep clearance at both ends: the head stops short of
1136    # the function glyph (>= side; extra room so it clears a subscript like
1137    # pow_{-2}), the tail leaves the argument glyph (<= side).
1138    _clear = "shorten >=2pt, shorten <=1pt"
1139    style = {
1140        "": "",
1141        "solid": f"[->, {_clear}]",
1142        "dotted": f"[densely dotted, ->, {_clear}]",
1143    }

1145        if w.kind == "segment" or w.kind == "pendant":
1146            lines.append(

1148                rf" -- ({endpoint(w.b, w.a)});"
1149            )
1150        elif w.kind == "loop":
1151            if w.a in group_side:
1152                pl, pr = group_side[w.a]
1153                lines.append(
1154                    rf"\path ({pl}) edge [out=160, in=20, looseness=1.6] ({pr});"
1155                )
1156            else:
1157                dist = ""
1158                if w.lane > 1:
1159                    dist = f", min distance={6 + 5 * (w.lane - 1)}mm"
1160                lines.append(
1161                    rf"\path ({name[w.a]}) edge [out=160, in=20, loop{dist}]"
1162                    rf" ({name[w.a]});"
1163                )
1164        elif w.kind == "arc":
1165            na, nb = endpoint(w.a, w.b), endpoint(w.b, w.a)
1166            tip = f", {w.direction}" if w.direction else ""
1167            dot = "densely dotted, " if w.arrow == "dotted" else ""
1168            if w.span <= 3:
1169                loose = 0.55 + 0.47 * w.span + 0.5 * (w.lane - 1)
1170                lines.append(
1171                    rf"\path[{dot.rstrip(', ')}] ({na}) edge [out=160, in=20{tip}, "
1172                    rf"looseness={loose:.2f}] ({nb});"
1173                )
1174            else:
1175                # long spans: flat north-anchored dome hugging the chain
1176                h = 0.35 + 0.08 * w.span + 0.28 * (w.lane - 1)
1177                arrowopt = f"[{dot}{w.direction}] " if (w.direction or dot) else ""
1178                lines.append(
1179                    rf"\draw{arrowopt.strip()} ({na}.north west) .. controls"
1180                    rf" +(-0.15,{h:.2f}) and +(0.15,{h:.2f}) .."
1181                    rf" ({nb}.north east);"
1182                )
1183        elif w.kind == "extra":
1184            a, b = w.a, w.b
1185            # start from the lower endpoint; bend so the curve bows DOWN,
1186            # away from the spine and its arcs
1187            if nodes[a].y > nodes[b].y or (
1188                nodes[a].y == nodes[b].y and nodes[a].x > nodes[b].x
1189            ):
1190                a, b = b, a
1191            bend = "right" if nodes[a].x < nodes[b].x else "left"

1193            tip = f"{w.direction}, " if w.direction else ""
1194            dot = "densely dotted, " if w.arrow == "dotted" else ""
1195            lines.append(

1197            )
1198        elif w.kind == "ring":
1199            lines.append(
1200                rf"\draw ({w.x + dx:.2f},{w.y:.2f}) circle (0.28);"
1201            )
1202        elif w.kind == "stub":
1203            d, anch = {
1204                "left": (f"({-STUB:.2f},0)", "west"),
1205                "right": (f"({STUB:.2f},0)", "east"),
1206                "up": (f"(0,{MIDSTUB:.2f})", "north"),
1207                "down": (f"(0,{-MIDSTUB:.2f})", "south"),
1208            }[w.direction]

1210                anch = {"west": "east", "east": "west",
1211                        "north": "south", "south": "north"}[anch]
1212            lab = ""
1213            if w.label:
1214                anchor = {"up": "south", "down": "north"}.get(w.direction, "west")
1215                lab = rf" node[anchor={anchor}, font=\scriptsize, inner sep=1.5pt] {{${w.label}$}}"
1216            src = name[w.a]
1217            if w.a in group_side:
1218                src = group_side[w.a][0 if w.direction == "left" else 1]
1219            shift = f"[xshift={w.x:.2f}cm]" if w.x else ""
1220            lines.append(rf"\draw ({shift}{src}.{anch}) -- ++{d}{lab};")
1221        elif w.kind == "bare":
1222            lines.append(
1223                rf"\draw ({w.x + dx:.2f},{w.y:.2f}) -- ++({1.0:.2f},0);"
1224            )
1225    _emit_boxes(layout, lines, prefix, name, group_side)

1257def _emit_derivs(layout: BookLayout, lines: list[str], prefix: str,
1258                 name: dict, group_side: dict) -> None:
1259    """Penrose derivative loops: fit-ellipse + boundary dot + whiskers."""
1260    nodes = {n.id: n for n in layout.nodes}
1261    for di, (enclosed, new_names) in enumerate(layout.derivs):
1262        eset = set(enclosed)
1263        parts = []
1264        for aid in enclosed:
1265            if aid in group_side:
1266                parts.extend(f"({p})" for p in group_side[aid])
1267            elif aid in name:
1268                parts.append(f"({name[aid]})")
1269        if not parts:
1270            continue
1271        # arcs/loops enclosed by the region must fit inside the ellipse:
1272        # include their topmost point in the fit spec
1273        for w in layout.wires:
1274            if w.kind == "arc" and w.a in eset and w.b in eset:
1275                h = (0.32 + 0.16 * w.span + 0.24 * (w.lane - 1)
1276                     if w.span <= 3 else 0.35 + 0.08 * w.span)
1277                mx = (nodes[w.a].x + nodes[w.b].x) / 2
1278                parts.append(f"({mx:.2f},{h + 0.12:.2f})")
1279            elif w.kind == "loop" and w.a in eset:
1280                parts.append(f"({nodes[w.a].x:.2f},{0.42 + 0.2 * w.lane:.2f})")

1282        # regions nest by creation order: inner derivatives walk first)
1283        inside = sum(
1284            1 for dj, (e2, _) in enumerate(layout.derivs)
1285            if dj != di and set(e2) <= eset and (set(e2) < eset or dj < di)
1286        )
1287        sep = 2.5 + 4.0 * inside
1288        en = f"{prefix}dE{di}"
1289        lines.append(
1290            rf"\node[ellipse, draw, inner sep={sep:.1f}pt, "
1291            rf"fit={{{''.join(parts)}}}] ({en}) {{}};"
1292        )
1293        if len(new_names) == 1:
1294            ang = 125 + 14 * inside  # fan nested whiskers apart
1295            lines.append(rf"\fill ({en}.{ang}) circle (1.4pt);")
1296            lines.append(

1298                rf" ++(-.24,.26)"
1299                rf" node[anchor=south east, font=\scriptsize, inner sep=1pt]"
1300                rf" {{${_tex_edge(new_names[0])}$}};"
1301            )
1302        else:
1303            ang = 55
1304            lines.append(rf"\fill ({en}.{ang}) circle (1.4pt);")
1305            whisk = [("80:.12", "++(-.2,.28)", "south east"),
1306                     ("45:.12", "++(.28,.18)", "south west")]
1307            for nm, (ctrl, end, anch) in zip(new_names, whisk):

1310                    rf" node[anchor={anch}, font=\scriptsize, inner sep=1pt]"
1311                    rf" {{${_tex_edge(nm)}$}};"

279            names = []
280            for e in new_edges:
281                tok = fresh("w")
282                g.deriv_tokens.add(tok)
283                out[e] = tok
284                names.append(e)
285            g.derivs.append((enclosed, names))
286            return out
287        if Expectation and isinstance(t, Expectation):
288            first_atom = len(g.atoms)
289            sub = walk(t.tensor)
290            enclosed = list(range(first_atom, len(g.atoms)))
291            g.boxes.append((enclosed, "E"))
292            return sub
293        if isinstance(t, Sum):
294            # a Sum factor becomes a parenthesized group atom; its inner
295            # terms are laid out recursively at layout time
296            aid = len(g.atoms)
297            toks = [fresh("w") for _ in t.edges]
298            atom = AtomSpec(aid, "group", "", toks, sub=t)
299            atom.port_names = dict(zip(toks, list(t.edges)))
300            g.atoms.append(atom)
301            return dict(zip(list(t.edges), toks))
302        raise NotImplementedError(f"book_layout does not support {type(t).__name__} yet")
303
304    exposed = walk(tensor)
305    for ta, tb in identity_pairs:
306        uf.union(ta, tb)
307    g.arrows = {uf.find(k): v for k, v in g.arrows.items()}
308    g.arrow_heads = {uf.find(k): v for k, v in g.arrow_heads.items()}
309    for atom in g.atoms:
310        atom.port_names = {uf.find(k): v for k, v in atom.port_names.items()}
311
312    # Build wires from union-find classes.
313    endpoints: dict[str, list[int]] = {}
314    for atom in g.atoms:
315        atom.ports = [uf.find(p) for p in atom.ports]
316        for p in atom.ports:
317            endpoints.setdefault(p, []).append(atom.id)
318    free_names: dict[str, str] = {name: uf.find(tok) for name, tok in exposed.items()}
319
320    for wid, ends in endpoints.items():
321        if len(ends) > 2:
322            raise ValueError("wire with more than two endpoints (use Delta)")
323        g.wires[wid] = ends
324    for name, wid in free_names.items():
325        if wid in endpoints:
326            g.free_wires.setdefault(wid, []).append(name)
327        # a wire with no atoms at all: bare identity between two free ends
328    # bare identity wires: group free names by wire id with no endpoints
329    deriv_reps = {uf.find(tok) for tok in g.deriv_tokens}
330    bare: dict[str, list[str]] = {}
331    for name, wid in free_names.items():
332        if wid not in endpoints and wid not in deriv_reps:
333            bare.setdefault(wid, []).append(name)
334    for wid, names in bare.items():
335        if len(names) == 2:
336            g.bare_wires.append((names[0], names[1]))
337        else:  # a dangling free edge with no atom: nothing to draw it from
338            raise ValueError(f"free edge {names} attached to nothing")
339    for tok in g.deriv_tokens:
340        if uf.find(tok) in endpoints:
341            raise NotImplementedError(
342                "a derivative's new edge is contracted onward; simplify() "
343                "first (drawing wires from the loop boundary is not "
344                "supported yet)"
345            )
346    # closed rings of pure order-2 identities: no atoms, no free names --
347    # invisible to the loops above, but each class is a scalar circle
348    ring_classes = set()
349    for ta, tb in identity_pairs:
350        rep = uf.find(ta)
351        if rep not in endpoints and rep not in set(free_names.values()):
352            ring_classes.add(rep)
353    g.rings = len(ring_classes)
354    return g
